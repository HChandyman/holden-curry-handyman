# Holden Curry Handyman Services

This is a simple static GitHub Pages website for Holden Curry's digital business card.

## How to use

1. Create a free GitHub account.
2. Create a new repository named `holden-curry-handyman`.
3. Upload these files:
   - `index.html`
   - `style.css`
   - `holden-curry.vcf`
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / root
6. Save.

Your website will publish at a link like:

`https://YOURUSERNAME.github.io/holden-curry-handyman/`

Write that URL to the NFC tag.
